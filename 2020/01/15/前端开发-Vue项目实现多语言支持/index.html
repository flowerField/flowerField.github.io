插件介绍

介绍：Vuei18n 是 Vue.js 的国际化插件。它可以轻松地将一些本地化功能集成到你的 Vue.js 应用程序中。

安装：npm install vue-i18n 或者 yarn add vue-i18n

项目使用

在项目目录下面创建一个文件夹(随便叫啥名都行)，这个文件夹主要用于处理i18n模块，做两件事情。

第一件事情是需要根据vue-i18n模块来导出一个i18n的实例对象，以供项目中能够直接使用。
