command   +   C 复制 command   +   V 粘贴 command   +   X 剪切 command   +   D 复制当前行或选中的区块 command   +   F 在当前文档执行搜索(查找) command   +   G 跳转到文档的某处 command   +   P 显示方法(函数)的参数信息 command   +   R 在当前文档执行替换操作 command   +   W 关闭当前选项卡 command   +   E 弹出最近文件，支持选择后打开 command   +   B 跳转到变量声明处 command   +   L 跳转到指定行(输入) command   +   Y 浮动窗口显示变量声明行
