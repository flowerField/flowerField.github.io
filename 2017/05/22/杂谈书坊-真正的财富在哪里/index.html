杂谈书坊-“我们真正的财富在哪里”

May 22, 2017 ✧ 字数统计:2k(字)    ♨︎ 阅读时长:6(分钟)

鼠年，这是一个很特别的年份，有些事实在发生，有些思考在变化。

最近，有很多迹象表明我该写这篇文章，用来总结、反省、选择或者是自我追问。

1991年10月出生的我，马上三十岁了。我从2010年起就开始写代码或者写作，大学期间学位修的是软件工程，通过在键盘上敲敲打打来工作或者是生活已逾十年，我的职业或者工作主要是软件开发或者是技术讲师。这十年里，我已放弃了成为一位作家的可能，或者也正是这种主观意愿的放弃让我能够陆陆续续在博客园和当前这个博客写下超 60W 字的技术分享文章(教程)。在这样的时间节点，似乎最适合回首过去和展望未来，不过我并不打算这么做，我可能更愿意多做一些思考，我的生活有很多的碎片，我希望能够通过这些碎片来更好的认清自己，或者想明白我的财富到底在哪里？

“你这几年都不快乐。”

有一天晚上，何芸跟我谈起了这个话题。其实我们彼此都心知肚明，这几年我内心没有快乐，大家也快乐不多，只是我们很少这样谈论。这几年我们从深圳到广州，工作起起伏伏家里又七零八落遭遇了很多，整体只能算凑合。唯一令人欣慰的是，何芸的事业算是稳中有升，只是遗憾加班太多。

我为什么不快乐？ 或者什么时候快乐过？不好说，也说不清。

我们得到的唯一一致的结论是：『在小码哥工作的第一年我是累并快乐的』,至于为什么，大概是那个时候我们都还很年轻，拿着当时相对很高的薪水未来有着很多种可能，更重要的是我当时很确信自己应该能够成为一名顶级的技术讲师，并且正走在看似正确的路上。还记得，在2015年我从深圳慧择离职的时候，有跟当时的人力资源总监熊姐讨论过放弃写作和离职的原因，我当时认为自己放弃写作是因为我认怂了，关键点在于当时的我觉得做报刊编辑或者主编，给报刊杂志约稿写文章或者在盛大、红薯网、榕树下这些平台写网文无法支撑起我以后的生活，而无疑我的自知之明告诉我自己很难成为余华或者是韩寒那样一流的作家。相较而言，我已经接受大学软件工程专业课程四年的训练，而且12、13年很顺利的拿到软考软件设计师和信息系统项目管理师证书，让我觉得自己觉得成为一位专家级的软件开发工程师可能会更容易些。

当初为什么加入『[在小码哥]』? 因为 MJ 老师的优秀以及我们间特殊的关系 ，再加上我可以在尝试跨行业工作挑战的同时还能够继续写代码。进入一个新的领域，让我能够学到和掌握很多新鲜的东西，比如如何把某个知识点真正的讲解清楚,如何给140多人上课并进行职业规划，如何管理工作团队等等。这样的平台也允许我把自己的才能展示和传递给更多的人，有压力有成长有期许而且还年轻，这就是理想中的工作和生活，我想这也正是累并快乐的原因吧，可惜好景不长。

然后，就有很多东西开始慢慢瓦解了，我对世界对人性等也都有了更多的认识。

我开始变得疲惫、怀疑和不信任，我甚至不知道自己是否达成了目标是一位一流的开发者或者是技术讲师，因为有些东西烟消云散，有些人无问西东，而是否达成了目标或者处于这一位置反而成为了最不重要的问题，甚至工作本身也变得不那么重要。有段时间，我总是长久的在思考生活更深层次的意义是什么，因为在这个社会无论你如何努力看起来都是在挣扎。既然如此，那么人生的终极意义是什么？我们在追求的究竟是什么？这样的想法很危险，很难长时间的保持克制。

有很多个跑步的清晨和夜晚，我沉浸到自我的思考中，我感受到自己很危险，不论是当前的状态、某些奇怪的想法、对未来的期许还是当前所处的位置。我积极的想要改变这种境遇，却发现自己陷入到了工作和生活划定的循环里，既无法跳出也无法到达。这种感觉，就像是我存在于一个 forEach 循环中，既无法通过任何语句和行为跳出，也无法预知究竟何时才能结束并终止循环，只能任凭太阳东升西落，一点点熬尽生命的残余。如果是熟悉 JavaScript 语言的朋友，我想应该特别能够感同身受。

我从念初中开始就养成了阅读的习惯，我也喜欢买书和屯书，在我租住的三室一厅小区房里，其中有整整一间房用来堆放着我读了很多遍的或者还尚未开封的数百本书，囊括了孔孟之道、经济、历史、文化、小说、艺术和技术等诸多类目，我阅读或者注视它们也仅仅能够获得一些片刻的安慰。说实话，兴许是阅读太泛的原因，我已经对那些不同高深的理论、新鲜的技巧、中庸的言论、看似起伏跌宕坎坷唏嘘的命运麻木了，我看任何故事都能够像个旁观者一样冷血和理智，我用心把这些内容插入到自己的命运片段去感受，却好似泥牛入海甚至打不起来一点点水花。

难以从自己书架中寻找到答案后，我开始把目光转移到更大的书架。想来十分可笑，可能这就是书呆子最后的倔犟，我去了广州图书馆，然后是体育中心的广州购书中心(新华书店)，我一层层的搜寻，上上下下的扫描。如果你刚好在场，那么或许你就会看到个神情沮丧，四处张望又两手空空的男子，因为疫情期间找不到理发师的原因，他还剪了个光头板寸的头发，这个人就是我。

我就是那个两手空空，又一无所获的人。

终于，在末了临走的时候，在文学类书架中我看到了一本新印刷的《没有一条路是重复的》，作者余华。余华 几乎是今日份的文坛中我最喜欢的中国作家，我甚至认为他比莫言更有资格获得诺贝尔文学奖。我并不喜欢讨论先锋文学的特点抑或是《活着》、《许三观卖血记》 和 [《在细雨中呼喊》]这些伟大的作品，因为它们实在是太重要了，也正因为如此所以也就没有了讨论的意义。

离开书店的时候，我买下了《没有一条路是重复的》，并一路看完第一部分，距今已经过去了很多天。
